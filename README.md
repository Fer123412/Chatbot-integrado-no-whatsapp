<h1 align="center">Chabot ( Botizinho )</h1>

> Status do Projeto: concluido :heavy_check_mark:

<p align="justify">Esse projeto consiste em três versões do mesmo bot tendo em cada uma melhoria partindo do terminal para integração no whatsapp:</p>

<ul>
  <li>Bot v1: simples</li>
  <li>Bot v2: usando modelo de IA</li>
  <li>Bot v3: integração Whatsapp</li>
</ul>

<h2>Botizinho v1: chatbot simples</h2>
<p align="justify">Com um codigo pouco mais simples e padrão de bot, você conversa normalmente com o bot so que com limitações de frases tendo uma lista de frases feitas que serão usadas, porém ele vai aprendendo sozinho enquanto conversa com você.</p>
